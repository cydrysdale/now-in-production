# Approved Examples

Use this to document annotated gold examples for each channel. These are pattern references — not templates to copy.

## Filling Out This Template

If this template has blank fields, help the user complete it before doing any other work. Use the interview guide below.

If this template is already complete, skip the interview and use it as campaign context.

### Interview Guide

Ask questions in small groups. Wait for answers before continuing. Use plain language.

**Start with:** "I'll help you build the approved examples file for this campaign. I'll ask about past work the team was happy with for each channel, and help you annotate what made it work. Let's start."

**Group 1 — Channels and scope:**

- What campaign is this for?
- What channels are in scope for this campaign?
- For each channel, do you have a past example the team considered successful?

**Group 2 — For each example the user identifies, ask:**

- What was the asset? Can you paste or describe it, or point to where it lives?
- Who was the audience and what was the objective?
- What specifically worked about this example? What made the team happy with it?
- What patterns from this example should future copy emulate? (e.g., how it opened, how it used proof, how it handled the CTA)
- What patterns from this example should future copy not copy? (e.g., a reference that was time-specific, a structure that only worked for that campaign)
- Is this example still current, or has the product, audience, or strategy changed since it was written?
- Who approved this example for reference use?

**Group 3 — Anti-patterns:**

- For any of the channels in scope, are there past examples or patterns the team considers bad — things that looked fine but performed poorly or needed heavy editing?
- For each anti-pattern: what was the issue, why did it fail, and what specifically should not be repeated?

**Group 4 — Gaps:**

- Are there any channels that need examples but do not have one yet?
- Are there channels where the team has not agreed on what good looks like?

After the interview, output the completed file using the template format below. Flag any channels without examples in Open Questions.
