# Editor Feedback Log Template — Draft

> **Status: Draft exploration.** Part of an active workflow exploration being refined through end of March 2026.

Use this file to turn editorial pain into packet improvements.

If the same issue appears twice, update the packet rather than fixing it only in-line.

Primary owner: copyeditor logs the issue, Production updates the packet, and the copywriter reviews strategic implications where applicable.

## Issue Log

For each meaningful rewrite, capture:

- Date:
- Campaign:
- Channel:
- Draft version:
- Copyeditor:
- What was rewritten:
- Why it was changed:
- Root cause:
  - missing strategy or task definition
  - wrong tradeoff or escalation rule
  - missing editorial standard or style drift
  - missing claim approval
  - weak example set or mimicry problem
  - incomplete channel spec
  - weak QA or acceptance criteria
  - other
- Packet file that should be updated:
- Specific update required:
- Copywriter review needed:
- Production update owner:
- Resolved:

## Recurring Issues To Watch

- polished but strategically off
- unsupported claims
- stale naming
- generic workflow tie-ins
- channel drift from the spine
- too much mimicry of old examples
- copy that is grammatically clean but off-brand
